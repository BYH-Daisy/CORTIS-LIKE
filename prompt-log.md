## 第 10 轮：歌词时间戳纠错 + 跳动修复

**用户：** 歌词文件有 3 处时间戳写错了，我排查出来了：结束早于开始、缺前导零、开始时间混段。

**助手：** 在源文件和项目文件中同步修正 3 处：

- `[02:05:13][01:08:26]` → `[02:05:13][02:08:26]`（结束时间早于开始）
- `[1:48:21]` → `[01:48:21]`（缺前导零）
- `[01:21:01][02:26:02]` → `[02:21:01][02:26:02]`（开始时间混在 2 分钟段）

**用户：** 歌词为啥有时候在两句的间隔中会跳歌词，滚动歌词没同步修改？

**助手：** 修复两个问题：

- 切歌时旧歌词数据未立即清空 → 加了 `setLyrics([])` + `setCurrentLyricIndex(-1)`，避免旧回调用上一首歌的时间戳匹配当前音频
- 滚动直接赋值 `scrollTop` 导致瞬间跳转 → 改为 `scrollTo({ top, behavior: 'smooth' })` 平滑滚动

## 第 12 轮：跨路由音频持续播放

**用户：** 点进成员简介页的时候音乐也要继续播放；从成员页返回时也要继续播，且页面不要刷新。

**助手：** 根本问题是 audio 元素和所有播放状态都在 `App.jsx` 内，路由切换时 `App.jsx` 卸载导致音频中断。

解决方案：

- 创建 `src/AudioContext.jsx`，将 `audioRef`、`currentSong`、`isPlaying`、`lyrics`、`currentTime`、`duration` 等状态以及 `handlePlay`、`switchSong`、`handleSeek`、`formatTime` 等函数提升到 Context 中
- 在 `main.jsx` 中用 `<AudioProvider>` 包裹 `<HashRouter>`，确保音频生命周期跨越所有路由
- `Carousel` 的 `onActiveIndexChange` 改为调用 `switchSong`（静默切换音频源，不自动播放）
- 修复 `Carousel` 挂载时触发 `onActiveIndexChange(0)` 导致切歌的问题：用 `isInitialMount` 跳过首次触发
- 修复 `Carousel` 返回后跳到第一首歌的问题：根据 `currentPlayingIndex` 计算初始 `position`
- 移除 `StrictMode`，避免开发模式下组件双重挂载导致 audio 元素被重建

## 第 14 轮：页面水印

**用户：** 把一个图案作为页面水印，浅灰色，封面不要水印，其他页面可以有，成员介绍页不要有，密度大小控制好。

**助手：** 图片放入 `public/` 并重命名为 `watermark.png`。经历多轮迭代：

1. **CSS `::before` 伪元素 + `z-index: -1`** → 被各 section 的渐变背景完全挡住，不可见
2. **固定定位 `div.watermark` + `background-repeat: repeat` 网格平铺** → 太整齐像网格，用户不满意
3. **`::before` + `rotate(12deg)` 旋转网格** → 仍然是网格规律排列
4. **多层 `background-image` 散乱分布** → CSS 无法独立旋转每层，且仍是固定分布
5. **JS 生成 `<img>` 元素，随机位置 + 随机旋转** → 完全散乱，但出现重叠
6. **网格分区随机偏移 + 取消旋转** → 均布不重叠，但密度需调整
7. **最终方案：** `useMemo` 生成 5×3 → 6×3 网格，每个格子内 50% 随机偏移，200px 等大，正向不旋转，`opacity: 0.06`，`z-index: 1` 浮于内容之上

关键实现：

- `Watermark` 组件在 `App.jsx` 内，用 `useMemo` 在挂载时生成固定随机坐标
- `pointer-events: none` 不干扰交互
- Cover 区 `z-index: 2` 盖住水印，成员页不渲染 Watermark 组件

## 页面结构

```
首页（/）
├── Cover 区     → 封面大图 + BlurText 入场文字
├── Sprout 区    → BubbleMenu 成员导航（MARTIN/JAMES/...）
├── Music 区     → 3D Carousel 歌曲轮播 + 歌词面板 + 进度条
├── Danmaku 区   → 幻灯片轮播 + 弹幕层 + 点赞
└── Footer       → 版权信息
```

成员页（`/martin`, `/james`, ...）→ 全屏背景 + 弹出动画 + 返回按钮
